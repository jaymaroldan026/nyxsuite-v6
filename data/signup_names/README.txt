Per-profile signup name lists
=============================

Put one name per line in each profile file.

Rules:
- Blank lines are ignored.
- Lines starting with `;` are ignored.
- Lines starting with `#` are ignored.
- Files are read as UTF-8, so emoji are supported.

Example:
alicia
alicia 💋💗
ur alicia 🧡
