Setup & Install

This opens the install web UI. If the Nyx Suite bridge isn't running yet, start it below — on a brand-new PC, do the one-time launcher step, then the bridge installs everything and the dashboard opens with a live install log.

One-time setup (fresh machine)

Extract the release folder, then run the launcher for your OS once. It creates the Python environment, installs dependencies + the Playwright browser, registers the agent, and starts the bridge. After that, this button does everything for you.

Windows

1Right-click run_nyx_suite.bat → Run
2or: PowerShell → .\portable_launch_nyx.ps1

macOS

1chmod +x *.sh *.command
2./run_nyx_suite.command

Linux

1chmod +x *.sh
2./run_nyx_suite.sh

Still seeing "native messaging host not found"? Close the bridge and run once: python agent_host/install_host.py --register then relaunch. After the bridge has run once, the “Start the bridge” button above works on its own.
